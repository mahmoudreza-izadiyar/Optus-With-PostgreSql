CRUD

This project wants to show CRUD functions. With Python Language and django framework. for storing data we use postgresql as SQL system to RDS.

READ

for this project we imagine that we want to store Car information into the Backend. each car has its own name, color, model and picture. In Home page ( index.html ). we will have all objects in a list. (READ)

DELETE

each car will shoe in a card, and each card has its own two button. Delete and edit if user want to delete an object, just by click on delete button this object will remove from database. ( DELETE )

UPDATE

if user click on edit, user will direct to a new page to edit information about the object,(UPDATE)

CREATE

We have a form at the top of the page, you can add new car object by fill the form and submit the form. (CREATE)

PostgreSQL CREATE PROCEDURE

Django will handel all coding of PostgreSQL create procedure by itself. the only database that let us to write by our self is oracle. but i want to show how would i write PostgreSQL for our database: to create Table of our database :

CREATE TABLE public."Cars"

(

id integer NOT NULL DEFAULT nextval('"Cars_id_seq"'::regclass),

name character varying(30) COLLATE pg_catalog."default" NOT NULL,

color character varying(30) COLLATE pg_catalog."default" NOT NULL,

model character varying(30) COLLATE pg_catalog."default" NOT NULL,

photo character varying(100) COLLATE pg_catalog."default" NOT NULL,

CONSTRAINT "Cars_pkey" PRIMARY KEY (id)

)

TABLESPACE pg_default;

ALTER TABLE public."Cars"

OWNER to postgres;
